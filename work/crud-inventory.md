### CRUD Inventory for GameSummary

- **Create**: Create a new GameSummary when a game is started successfully and the initial Game instance is returned from the web service.
- **Read**: Retrieve all GameSummary records to display a list of games that are either in progress, completed, or expired/deleted.
- **Update**: Update a GameSummary everytime a new guess is submitted so that the guess count, status, and last guess timestamp remain up to date.
- **Delete**: Remove GameSummary entries that the user selects from a list and elects to delete.
